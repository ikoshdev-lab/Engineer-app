# Engineer App 🚀

O'zbekistonlik dasturchilar va muhandislar uchun professional ijtimoiy tarmoq platformasi.

## 🌟 Imkoniyatlar

- **Professional Profil:** O'z ko'nikmalaringiz, tajribangiz va portfoliongizni namoyish eting.
- **Ish Qidirish:** Mahalliy va xalqaro IT vakansiyalarini toping.
- **Jonli Efir (Live):** Bilim almashish uchun vebinarlar o'tkazing.
- **Chat va Guruhlar:** Boshqa mutaxassislar bilan real vaqtda muloqot qiling (ovozli va video xabarlar).
- **Kod Muharriri:** Brauzerning o'zida kod yozing va ulashing.
- **Mobil Ilova (PWA):** Telefonda ilova kabi ishlaydi.

## 🛠 Texnologiyalar

- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (Vanilla)
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Real-time:** Socket.io (Chat va Bildirishnomalar uchun)
- **Media:** WebRTC (Video qo'ng'iroqlar)

## 🚀 Ishga tushirish

Loyihani kompyuteringizga yuklab olish va ishga tushirish uchun:

1. **Repozitoriyani yuklab oling:**
   ```bash
   git clone https://github.com/ikoshdev-lab/Engineer-app.git
   cd Engineer-app
   ```

2. **Kerakli kutubxonalarni o'rnating:**
   ```bash
   npm install
   ```

3. **Serverni ishga tushiring:**
   ```bash
   node server.js
   ```

4. **Brauzerda oching:**
   `http://localhost:5001`
